# 表格标签
  * 主要用于规则的数据显示
  * 适当的可以在表单页面中使用
   >缺点：* 代码量比较大，页面浏览速度比较慢
          * 层次结构比较复杂，不易于维护
          * 不利于搜索引擎搜索数据

###表格相关元素
  * table 用于定义表格
  * cellspacing  单元格的内边距
  * cellpadding 单元格的外边距 
  * tr定义表格行，该元素只能包含<td>或者<th>两种元素
  * td 定义单元格，包含两个主要的属性：
  * colspan 指定单元格可以跨越多少列
  * rowspan 指定单元格可以跨越多少行
  * aption 用于定义表格标题
  * th 定义表格页眉的单元格
  * tbody 定义表格的主体
  * thead 定义表格头
  * tfoot 定义表格脚
